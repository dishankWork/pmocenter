Tech Architecture

Technology stack powering the PM Command Center

🔐 Auth Layer
🔑
bcryptjs
Password hashing with salt rounds — secure credential storage
🎫
jsonwebtoken (JWT)
Stateless auth tokens stored in httpOnly cookies — 8hr expiry
🍪
cookie-parser
Secure httpOnly, sameSite:lax cookie handling
👤
RBAC Middleware
Role-based access: program-director, cxo, pm, client
🔌 API Layer
🚀
Node.js + Express
RESTful API with JSON responses — no build step required
📄
pdfkit
A4 PDF generation with branded headers, tables, and charts
📊
exceljs
Multi-sheet Excel exports with styled headers and colored data
🛡️
Auth Middleware
All /api/projects and /api/export routes protected by JWT
🗄️ Database Layer
💾
sql.js (SQLite)
Pure JS SQLite — no native bindings, persisted to data/pm.db
🌱
Auto-seeding
Seeds 6 users + 5 projects with full data on first run
📝
9 Tables
users, projects, risks, actions, highlights, nextsteps, pain, billing, issues
🔄
Replace Pattern
Child records deleted + re-inserted on every PUT for simplicity
🎨 Frontend Layer
📱
Vanilla HTML/CSS/JS
Single-page app — no React, no build tools, no frameworks
🔤
Inter + Libre Baskerville
Google Fonts — Inter UI, Libre Baskerville for display headings
🎯
Design System
CSS variables, status pills, cards, grids — consistent tokens
🔒
RBAC Frontend
_permissions from API drives show/hide of sensitive sections
